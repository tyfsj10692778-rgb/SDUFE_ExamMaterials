# 山东财经大学期末复习资料整理

本仓库用于整理山东财经大学各专业、各课程的期末复习资料，内容包括历年试题、复习重点、课堂课件、题库、笔记、思维导图等，方便学弟学妹们在期末复习的时候快速查找使用。

资料由个人收集、整理和归档，欢迎同学们按需参考，也欢迎补充和完善。

## 资料分类

目前资料主要按学科方向分类：

- `计算机类`：程序设计、数据库、数据结构、计算机网络、数字图像处理、数据可视化等
- `经济金融类`：微观经济学、宏观经济学、金融学、统计学、财税法等
- `数学大类`：微积分、高等数学、线性代数、概率论、离散数学、大学物理等
- `政治文字类`：思政、近代史、毛概、习概、大学英语、军事理论等
- `会计管理类`：会计、管理学等

## 使用方式

如果你不熟悉 GitHub，推荐直接下载整个资料包：

1. 点击页面右上角绿色的 `Code` 按钮
2. 选择 `Download ZIP`
3. 下载完成后解压，在文件夹里按课程名称查找资料

也可以直接在 GitHub 页面中点开对应目录，找到自己需要的课程资料后单独下载。
